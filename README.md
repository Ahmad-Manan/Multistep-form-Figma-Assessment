## Features & Implementation

- Fully **responsive layout** for desktop, tablet, and mobile.  
- Optimized **images and assets** for fast loading by uploading it as a webpg or SVG.  
- Multi-step form with **custom styling** matching the Figma design.  
- Form emails are fully functional using **WP SMTP setup**.  
- Implemented in a **WordPress environment** using a child theme of choice.  
- Clean and modular **HTML, CSS, and PHP** code for maintainability.

---

## Custom CSS

All additional styling to match the Figma design is included in `plugins/simple-css`.  

This includes:

- Typography adjustments  
- Spacing and margins  
- Colors and backgrounds  
- Multi-step form specific styling (buttons, inputs, step indicators)  

> Note: The CSS was also applied live via the **Simple CSS plugin** in WordPress. For GitHub, all CSS is included in the exported file so it can be reviewed easily.
